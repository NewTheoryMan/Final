Azure



Definition

Windows Azure is an open and flexible cloud platform that enables you to quickly build, deploy and manage applications across a global network of Microsoft-managed datacenters.

Impression

Azure is a SaaS and PaaS platform that allows for cloud hosting of not only websites, but repositories, and SQL databases. It offers scalablity and various tools integration. It has analytical tools for understanding the big data message behind your database or the visitors to your website.

Take Away

I liked using Azure, but quickly found that many people in the class had trouble publishing their websites and repositories to it. It is also a fairly expensive service at $200/year. That doesn't seem like a lot of money for someone who uses it all the time, but it's definitely too much for a transitioning Soldier on a student budget. I would have definitely liked to work with it more, but found that I prefer Github and Bitbucket for storing my repositories.
