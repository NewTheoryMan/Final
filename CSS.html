Cascading Stylesheets



Definition

CSS is a style sheet language used for describing the presentation semantics (the look and formatting) of a document written in a markup language.

Impression

CSS makes editing stylistic elements of a website incredibly simple. I found the examples of the CSS Zen Garden to be incredibly clever use of CSS and how important it is to give your website the right look and functionality. CSS is vital for making sure a website is not only attractive enough to warrant its use, but functional, as it contains many features that HTML cannot do alone.

Take Away

I esecially enjoyed getting to learn more about CSS and how it has more to offer in terms of functionality. I had no idea that I could created fully interactive menus and charts with only CSS. I think that some very talented people create the best content on the web and changing their style is as simple as tweaking a few lines of code, rather than adjusting each individual class or id. I want to get more familiar with advanced CSS after I leave this course.
