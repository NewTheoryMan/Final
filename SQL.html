Definition

Microsoft SQL Server is a cloud-ready information platform that will help organizations unlock breakthrough insights across the organizations and quickly build solutions to extend data across on-premises and public cloud.

Impression

SQL is fascinating to me because as the internet continues to grow, so too does the amount of data stored in various disjointed databases around the world. I'd like to hope that one day we can harness all data and link it in an intuitive way that can lead to improvements in our way of life and scientific discovery. Getting to see the "behind the scenes" of how developers create databases and store data was amazing to me because most of my experience has been on the user end, working the the GUI of a final product.

Take Away

Because of my strong background in data mining and data analysis, I think SQL is going to be my primary focus after this course. I want to hone my knowledge and keep learning about software and tools that interests me and has made my job a lot easier.
