Hypertext Markup Language



Definition

HTML is a language for describing web pages. It has tags that are used to literally markup a web page by arranging the elements within it.

Impression

I absolutely love working with HTML. I feel like I get to be an artist who arranges different pieces to make an interesting collage of informatinon. If I don't have what I need to create a certain effect, I can either browse the plethora of help websites, or do some interesting work-arounds that give me my desired result. It was great to work with HTML again, and I definitely feel like it has been one of my strong points in this course.

Take Away

I learned quite a bit while reteaching myself HTML. I liked hearing about the many new features of HTML5, but I liked even more getting to take my basic knowledge a couple steps forward by getting better at integrating CSS and Javascript into my web pages.
