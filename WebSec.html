Web Security



Definition

Web Security is a branch of computer security specifically related to the Internet, often involving browser security but also network security on a more general level as it applies to other applications or operating systems on a whole.

Impression

As a practitioner of information security and former crypto-key custodian, I was very familiar with the concepts of security and risk in the world of computing, as well as the internet. The introduction of new forms of currency like Bitcoin presents new challenges to those concerned for the safety of not only the economy, but the entire internet. I was thrilled to be able to create my own security certificate, and it was interesting to learn how to register it and incorporate it in my future web projects.

Take Away

Cyber security is a growing field and will likely remain important as long as the internet remains free and open. I want to continue to stay on top of current trends in the cyber-security realm and learn some of the actual techniques that keep our computers and computer networks safe.
