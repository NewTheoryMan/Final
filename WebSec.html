Web Security



Definition

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Impression

What I learned the most about Git is to not be afraid of an application's interface, just because I am unfamiliar. I was initially afraid of working with anything that looked like a console window because of my past experiences with MS-DOS. I had a series of bad experiences with the MS-DOS prompt when I was younger, and Git looked very intimidating. I found, however, that all I needed was the basic commands and Git was very simple to use. I could create directories, make files, initiate a repository, keep my files up-to-date, and quickly upload the project to Azure or Github.

Take Away

What I plan to take away is my knowledge of the software and concepts of version control, but also increased confidence when dealing with unfamiliar software. I plan to keep using Git, however frequently I can manage, and learn how to create and manipulate branches.
